# HardSecNet Hardening Studio - Design System

This document outlines the core color palette and typography extracted from the HardSecNet Hardening Studio application.

## 🎨 Color Palette

### Base Backgrounds
- **Main Application Background:** `#eef3f8` (Light grayish-blue)
- **Sidebar Background:** `#0e1726` (Deep navy blue)
- **Navigation Menu:** `#0b1322` (Very dark blue)
- **Cards & Panels:** `#ffffff` (Pure white)

### Text Colors
- **Primary Text (Dark):** `#182331`, `#0f172a` (Headings and main body text)
- **Primary Text (Light):** `#e8eef7` (Used on dark backgrounds like the sidebar)
- **Subtitles/Muted (Light):** `#b8c7d9` (App subtitle)
- **Subtitles/Muted (Dark):** `#5b6b7f`, `#5b6b7d` (Page subtitles, panel text)

### Accent Colors
- **Brand/Logo:** `#67e8d5` (Bright teal/cyan)
- **Primary Buttons & Active State:** `#f4c84f` (Yellow-gold) - *Hover: `#e8bb3f`*
- **Secondary Buttons:** `#1d2a3d` (Dark blue-gray) - *Hover: `#263956`*

### Status / Metric Card Accents
- **Cyan:** `#16a3b8` (Ready Actions)
- **Green:** `#2f9e6d` (Compliance Score)
- **Amber:** `#c18a00` (Open Findings)
- **Coral:** `#e05a47` (Drift Changes)
- **Violet:** `#7c5cff` (Reports Ready)
- **Blue:** `#2f75d6` (Last Run)

---

## 🔠 Typography

### Font Families
- **Primary:** `Arial`, `Segoe UI`, `Calibri`, `sans-serif` (The application dynamically checks system fonts and falls back gracefully).

### Hierarchy
- **App Title & Page Titles:** `27px`, `900 weight` (Extremely bold, prominent)
- **Metric Values:** `30px`, `900 weight`
- **Panel Titles:** `16px`, `800 weight`
- **Section Titles:** `15px`, `900 weight`
- **Base Body Text:** `13px`, normal weight (with `800 weight` used for labels and headers)
- **Metric Labels:** `13px`, `800 weight`
- **Metric Captions:** `12px`, normal weight
